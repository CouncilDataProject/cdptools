cdptools.research_utils.transcripts.download_transcripts(db: cdptools.databases.database.Database, fs: cdptools.file_stores.file_store.FileStore, order_by_field: str = 'confidence', save_dir: Union[str, pathlib.Path, None] = None) → Dict[str, pathlib.Path][source]¶

Download a transcript for each event found in a CDP instance. Additionally saves the manifest as a CSV.

Parameters

  • db (Database) – An already initialized database object connected to a CDP instance’s database.

  • fs (FileStore) – An already initialized file store object connected to a CDP instance’s file store.

  • order_by_field (str) – Which field to order the transcripts by to select the first (highest value) of. Default: “confidence” Choices: [“created”, “confidence”]

  • save_dir (Optional[Union[str, Path]]) – An optional path of where to save the transcripts and manifest CSV. If None provided, uses current directory. Always overwrites existing transcripts with the same name if they already exist in the provided directory.

Returns

event_corpus_map – A dictionary mapping event id to a local Path for a transcript for that event.

Return type

Dict[str, Path]
